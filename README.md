## Приложение для рецептов и списков покупок

В работе представлена программа на языке Python, благодаря которой можно работать с разными рецептами. Программа 
умеет создавать ингридиенты, объединять их в рецепты (обычные/диетические), а также впоследствии собирать их в списки 
покупок (с учетом количества порций). 

## Правила пользования
Перед запуском рекомендуется установить библиотеку для тестов `pytest`, можно сделать это через соответствующую команду 
в терминале: 
```bash
pip install -r requirements.txt
```
Все тесты находятся в файле test_recipes.py, для их запуска можно просто написать в терминале команду:
```bash
pytest
```
Если говорить про правила использования, то снизу указан пример работы со всеми классами:
```python
from recipes import Ingredient, Recipe, ShoppingList  # импорт классов
notSugar = Ingredient(name="Аспартам", quantity=2, unit="г")  # cоздание ингредиента 
dobri_cola = Recipe(title="КокаКоля", ingredients=[notSugar])  # создание рецепта
shopping = ShoppingList()  #  создание списка покупок
shopping.add_recipe(dobri_cola, portions=2)  # добавление рецепта в список покупок
```
